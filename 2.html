Command Descriptions

Navigation Commands:

cd [directory]: Change directory.
ls [options] [directory]: List directory contents.
pwd: Print working directory.
mkdir [directory]: Make directory.
rmdir [directory]: Remove directory.
cp [options] source destination: Copy files and directories.
mv [options] source destination: Move or rename files and directories.
rm [options] file: Remove files or directories.

File Manipulation Commands:

cat [file]: Concatenate and display file content.
more [file]: Display file content page by page.
less [file]: Display file content with backward navigation.
head [options] [file]: Output the first part of files.
tail [options] [file]: Output the last part of files.
touch [file]: Create an empty file or update timestamp.
chmod [options] mode file: Change file permissions.

Text Processing Commands:

grep [options] pattern [file]: Search for patterns in files.
sed [options] 'command' [file]: Stream editor for filtering and transforming text.
awk 'pattern {action}' [file]: Pattern scanning and processing language.
sort [options] [file]: Sort lines of text files.
uniq [options] [file]: Report or filter out repeated lines in a file.
cut [options] [file]: Remove sections from each line of files.
paste [options] [file1] [file2]: Merge lines of files.
tr [options] set1 set2: Translate or delete characters.

System Information Commands:

uname [options]: Print system information.
df [options]: Report file system disk space usage.
du [options] [directory]: Estimate file space usage.
top: Display Linux processes.
free [options]: Display amount of free and used memory in the system.
ps [options]: Report a snapshot of the current processes.
lsof [options]: List open files.

Network Commands:

ping [options] host: Send ICMP ECHO_REQUEST to network hosts.
traceroute [options] host: Print the route packets trace to network host.
ifconfig [interface] [options]: Configure network interface parameters.
netstat [options]: Print network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.
nslookup [options] [host]: Query Internet name servers interactively.

Network Monitoring Tools:

iftop [options]: Display bandwidth usage on an interface.
nload [options]: Monitor network traffic and bandwidth usage in real-time.
tcpdump [options]: Capture and display network packets.
iptraf [options]: Interactive IP LAN monitor.

Package Management Commands:

apt-get [options] [command] [package]: Command-line tool for handling packages.
dpkg [options] [package]: Package manager for Debian-based systems.
yum [options] [command] [package]: Package manager for RPM-based systems.
rpm [options] [command] [package]: RPM package manager.
pip [options] [command] [package]: Package installer for Python.
conda [options] [command] [package]: Package manager for Python environments.

User Management Commands:

adduser [username]: Add a user to the system.
userdel [username]: Delete a user account and related files.
passwd [username]: Change user password.
usermod [options] [username]: Modify user account.

Process Management Commands:

kill [options] [process ID]: Send signals to processes.
killall [options] [process name]: Kill processes by name.
nice [options] command: Run a command with modified scheduling priority.
renice [options] [priority] [process ID]: Alter priority of running processes.
jobs [options]: Display status of jobs.
